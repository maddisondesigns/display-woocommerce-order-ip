A simple plugin to display the Customer IP on the Orders List page.

## Description

Adds a column with the Customer's IP address onto the Orders List page. This will save you from editing each order just to view the IP addres of the customer, and especially useful when investigating failed or fraudulent orders.

## Installation

1. [Download latest zip from Github](https://github.com/maddisondesigns/display-woocommerce-order-ip/releases/latest)
2. Unzip file and upload the 'display-woocommerce-order-ip' folder to your '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
